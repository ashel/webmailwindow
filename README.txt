This program is to control Webmail window LED pattern.

usage:
% webmailwindow.exe -c [r|g|b|rg|rb|gb|rgb|none]

To build this program, you must install Windows Driver Kit.

After that, put project files to WinDDK\[Version]\src\hid\webmailwindow,
cd there, and execute

% build
